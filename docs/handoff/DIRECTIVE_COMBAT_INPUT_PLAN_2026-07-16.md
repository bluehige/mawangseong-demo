## 7. 미해결 항목과 위험

- Windows 한글 입력의 정확한 원인은 네이티브 빌드와 Microsoft 한국어 IME로 재현하기 전에는 확정할 수 없다.
- 입력 화면 재생성·포커스 재획득과 전역 키 라우팅이 우선 조사 대상이지만, 재현 로그에 따라 수정 지점이 달라질 수 있다.
- 직접 조작 제거는 UI 삭제만으로 끝나지 않으며 수동 스킬과 특수 유닛 행동을 자동 AI로 이관해야 한다.
- 충돌 제거와 x3를 동시에 적용하면 집중 화력과 연출 시간축이 바뀔 수 있어 단계별 기준선 비교가 필요하다.
- 화면마다 고정 좌표와 자동 폰트 축소 방식이 달라 한 해상도만 수정하면 다른 해상도에서 재발할 수 있다.
- 현재 세션에서는 어떤 구현·빌드·플레이 검증도 수행하지 않았다.
