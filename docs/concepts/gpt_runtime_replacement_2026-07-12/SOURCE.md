# GPT 내장 생성 런타임 그래픽 원본

이 폴더의 PNG 15개는 2026-07-12에 Codex의 내장 ImageGen으로 생성했다.

- 캐릭터 4×4 시트 8장: 슬라임, 고블린, 임프, 탐험가, 도둑, 수련생 용사, 방패병, 셀렌
- 진화 배지 3×2 시트 1장
- 기본 UI 아이콘 3×3 시트 1장
- 구형 방 소품 3×3 시트 1장
- 구형 타일 3칸 시트 1장
- 기본 VFX 4×6 시트와 선택 원 4칸 시트
- 방패병 전용 정사각 초상 1장

`tools/prepare_gpt_runtime_assets.py`는 이 원본을 자르고 자홍색 배경을 투명하게 바꾸며 게임 규격으로 축소한다. 새 그림을 직접 그리는 코드는 포함하지 않는다.

이 교체에서 제외한 항목은 현재 런타임 파일이 이미 GPT 내장 생성 원본으로부터 준비된 조사관, 공병, 로만, 레온, 진화형, 엔딩 일러스트, 스킬 아이콘, 진화 VFX, 던전 쿼터 v2 리소스와 게임에 쓰이지 않는 QA/컨셉 미리보기다.
