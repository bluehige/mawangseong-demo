# 3차 업데이트 Phase 4 완료 기록

작성일: 2026-07-13

## 결과

- 전선을 선택한 새 회차는 Stage 02부터 `heart_chamber` 1개를 고정 배치한다.
- 심장실은 5x5 방과 2x2 세로 통로로 병영에 연결되며, 입구→왕좌 기존 경로를 변경하지 않는다.
- Stage 02/03/04 체력은 각각 420/600/820이다.
- 심장실과 연결 통로는 이동, 연결 해제, 삭제, 시설 교체가 불가하다.
- 심장실 HP 0은 방 비활성으로만 처리하며 마왕성 패배나 왕좌 HP 감소를 발생시키지 않는다.
- 심장실이 비활성되면 적의 목표는 즉시 원래 왕좌로 복귀한다.
- 기존 v3 회차에는 심장실을 삽입하지 않는다.
- 최종 아트 전까지 `heart_core_placeholder`를 프로그램으로 그려 표시한다.

## 저장 계약

- v4 `heart` 영역에 `chamber_spawned`, `chamber_hp`, `chamber_max_hp`, `disabled_this_battle`를 보존한다.
- 배치되지 않은 방은 HP 0/0, 배치된 방은 최대 HP 1 이상을 강제한다.
- Stage 02 이상의 전선 선택 회차에는 심장실 배치를 강제한다.
