## 9. Phase 0 종료 판단

Phase 0은 아래 네 조건을 모두 만족하면 완료다.

- 마스터 명세와 이 매트릭스가 기준 SHA의 실제 구현을 반영한다.
- 1.2에서 보존할 것과 2.0에서 재설계할 것이 파일 경계까지 구분돼 있다.
- 코드, 데이터, 수치, 씬, 자산과 워크플로 변경이 없다.
- 사용자가 DAY 1~5 제품 계약을 승인하기 전 Phase 1을 시작하지 않는다.
