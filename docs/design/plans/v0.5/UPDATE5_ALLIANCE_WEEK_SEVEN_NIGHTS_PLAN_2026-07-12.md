## 0. Codex 실행 계약
이 문서는 5차 업데이트의 아이디어 목록이 아니라 구현 순서와 범위를 고정하는 작업 계약이다. 아래 규칙은 다른 모든 제안보다 우선한다.
1. 5차는 기존 30일 캠페인 뒤에 DAY 31을 추가하지 않는다.
2. 5차는 세 번째 장기 30일 캠페인을 만들지 않는다. 대신 50~75분 목표의 짧은 반복 모드 `alliance_week`를 추가한다.
3. `alliance_week`는 정확히 7개의 전투 밤으로 구성하고, 각 전투 사이에 짧은 생활·정비 단계를 둔다.
4. 5차에서는 신규 영구 플레이어블 종족을 추가하지 않는다. 로스터 포화보다 기존 몬스터의 애착과 최종 성장 보강을 우선한다.
5. 왕관 진화는 정확히 6종만 추가한다: 돌콩, 두둠, 루미, 미미, 베베, 코코.
6. 실키의 왕관 진화는 5차 범위에서 제외한다. 4차 실제 사용 자료와 문턱 제어 밸런스가 충분히 축적된 뒤 다시 판단한다.
7. 합동 기억과 합동기는 정확히 6개만 추가한다.
8. 신규 지역은 정확히 2개만 추가한다.
9. 신규 일반 적은 신규 지역용 4종만 추가한다.
10. 신규 전초기지는 정확히 2종만 추가해 기존 3종과 합계 5종으로 만든다.
11. 경쟁 마왕성 방문 도전은 브라사·베스퍼·미렐라 각 1개 기본 맵과 2개 시나리오만 만든다.
12. 방문 도전에서 경쟁 마왕을 풀 플레이어블 유닛으로 만들지 않는다. 전투당 1회 지원 명령으로 표현한다.
13. 최종전은 레온·셀렌·로만의 삼중 합동 공성이다. 세 보스를 동시에 상시 활성화하지 않고 단계별로 교대시킨다.
14. 마왕성 생활 애니메이션은 몬스터당 1개 서명 동작만 제작한다. 완전한 생활 시뮬레이션으로 확장하지 않는다.
15. 방 꾸미기는 고정 장식 앵커에만 배치한다. 충돌·경로·전투 능력치를 바꾸지 않는다.
16. 장식과 생활 장면은 전투 수치를 절대 제공하지 않는다.
17. 커스텀 도전은 도감에서 해금한 항목을 조합하는 제한형 제작기다. 자유 웨이브 스크립트 편집기는 만들지 않는다.
18. 커스텀 도전 코드는 오프라인 문자열 공유만 지원한다. 서버, 온라인 랭킹, 시즌 종료, PvP는 구현하지 않는다.
19. 주간 모드 전투 수치는 규격화한다. 기존 장기 세이브의 무한 레벨·자원 우위를 그대로 반영하지 않는다.
20. 주간 모드의 보상은 장식, 생활 기록, 도전 도장, 프리셋 같은 수평 보상만 제공한다.
21. 정상 난이도에서는 전투 직전 안전 저장으로 해당 밤 재도전을 허용한다. 재도전 금지는 선택형 도전 인장으로만 제공한다.
22. 주간 모드 패배는 몬스터 영구 사망, 유대 하락, 프로필 자원 손실을 만들지 않는다.
23. 신규 적은 특정 몬스터나 왕관 형태를 무효화하지 않는다. 대상 효율을 20~35% 낮추는 소프트 카운터로 제한한다.
24. 신규 엔딩은 E23~E28의 6개만 추가한다.
25. 기존 E00~E22를 삭제하거나 낮은 등급으로 취급하지 않는다.
26. 저장은 v6로 올린다. 진행 중 v5 캠페인에 주간 모드를 강제로 삽입하지 않는다.
27. `GameRoot.gd` 또는 `CombatSceneController.gd`에 주간 밤 번호·손님 성·지역·왕관 ID별 거대 분기문을 추가하지 않는다.
28. 새 모드와 도전 제작기는 데이터 계약과 순수 서비스로 먼저 검증한 뒤 UI에 연결한다.
29. placeholder 행동과 수치로 수직 구현을 통과한 뒤 최종 이미지·애니메이션·오디오를 제작한다.
30. 1920×1080과 1366×768 회귀 검사를 모두 유지한다.
31. 각 Phase 종료 때 변경 파일, 결정 로그, 자동 검사, 수동 캡처 결과를 남긴다.

## 1. 선행 조건과 시작점
### 1.1 4차 업데이트 완료 인정 조건
- 저장 v5와 v4→v5 마이그레이션이 무손실이다.
- `front_chronicle`과 `council_season`이 모두 DAY 30까지 완주된다.
- 마계 의회 회차의 지역 5개 중 3개 선택이 실제 웨이브·이벤트·보상에 반영된다.
- 전초기지 3종이 DAY 10·20 전투에서 서로 다른 결과를 만든다.
- 제한적 2층에서 계단 전이·상층 목표·비가시 층 전투가 정상 작동한다.
- 브라사·베스퍼·미렐라가 서로 다른 전투 규칙과 관계 결말을 갖는다.
- 왕관 진화 C01~C06이 회차당 1마리 제한, 거절 대체 보상, 도감 기록을 갖는다.
- 실키와 포포가 기존 몬스터의 상위호환이 아니다.
- E00~E22가 도달 가능하고 E17~E22 조건이 서로 충돌하지 않는다.
- 의회 회차의 평균 완주 시간이 목표 범위에 들어온다.
- 왕관 진화 선택률이 특정 1종에 60% 이상 몰리면 원인을 확인했다.
- 상층 레이아웃 하나의 선택률이 65% 이상이면 경로·목표 난도를 조정했다.
- 전체 자동·수동·UI·저장·성능 검증이 PASS다.

### 1.2 5차가 해결할 문제
1. 1회차가 30일이므로 다른 빌드를 시험하려면 여전히 긴 시간이 필요하다.
2. 로스터가 13종까지 늘었지만 한 회차에서 실제로 써 보는 조합은 일부에 머문다.
3. 왕관 진화가 6종뿐이라 오래 키운 나머지 몬스터는 최종 성장 종착점이 없다.
4. 경쟁 마왕과 관계를 맺었지만 실제로 그들의 성을 방문해 전투 규칙을 체험하지 못한다.
5. 합동 기억은 전투 조합을 만들지만 아직 절반의 몬스터 관계만 표현한다.
6. 전투 외에 몬스터가 성에서 살아가는 모습이 초상·대사에 비해 부족하다.
7. 도감이 커져도 플레이어가 해금 콘텐츠를 직접 조합해 시험할 공간이 없다.
8. 레온·셀렌·로만은 각각 강한 라이벌이지만 세 사람이 같은 작전표를 쓰는 최종 시험이 없다.
9. 전초기지와 지역 선택이 장기 캠페인 안에서만 쓰여, 짧은 반복 콘텐츠에서 재사용되지 않는다.
10. 메타 보상이 많아질수록 수치 성장보다 수평적 수집·기록·표현 보상이 더 필요하다.

## 2. 업데이트 정체성과 목표
### 2.1 업데이트 명칭
**5차 대형 업데이트: 연맹 주간과 일곱 밤의 성**

핵심 문장:

> “서른 날을 다시 살지 않아도, 일곱 밤이면 새로운 부하 조합과 새로운 성을 시험할 수 있다.”

### 2.2 플레이어 감정 목표
1. 긴 캠페인 사이에 부담 없이 새로운 로스터를 시험한다.
2. 브라사·베스퍼·미렐라의 성을 직접 지키며 각 마왕의 운영 철학을 체감한다.
3. 돌콩·두둠·루미·미미·베베·코코에게도 왕관을 씌워 오래 키운 보람을 느낀다.
4. 몬스터들이 전투가 끝난 뒤 쉬고, 일하고, 장난치는 모습을 보며 성이 실제 거주 공간처럼 느껴진다.
5. 도감에서 적·지역·맵·제약을 조합해 나만의 방어 시험을 만든다.
6. 레온·셀렌·로만의 세 전술을 한 전투에서 해석하는 최종 숙련 시험을 경험한다.
7. 한 주 결과 카드가 이전 주와 달라 다음 조합을 자연스럽게 계획한다.
8. 반복 보상이 공격력 증가가 아니라 표현과 선택 폭을 늘려 장기 세이브의 밸런스를 지킨다.

## 4. 연맹 주간 전체 루프
### 4.1 해금 규칙
- 마계 의회 회차를 한 번 이상 승리로 완료한다.
- `council_season_any_clear = true`를 프로필에 기록한다.
- 전선 연대기 회차는 연맹 주간 해금에 필수가 아니다. 인간 라이벌 관계 대사는 해당 기록에 따라 달라진다.
- 첫 해금 시 바티가 “서른 날짜리 보고서를 일곱 밤으로 줄였다”는 3줄 안내를 한다.
- 첫 주에는 주간 헌장 2개, 게스트 성 3개, 기본 전초기지 3개만 표시한다.
- 첫 주 클리어 뒤 신규 지역·전초기지·도전 공방을 단계적으로 해금한다.

### 4.2 시작 준비 4단계
| 단계 | 선택 항목 | 제한 |
| --- | --- | --- |
| 1. 전력 편성 | 몬스터 5종, 심장 1개, 왕관 후보 0~1, 합동기 0~2 | 동일 몬스터 중복 금지 |
| 2. 연맹 노선 | 게스트 성 1개, 전초기지 1개, 지역 카드 2개 | 게스트 성은 주간당 1곳 |
| 3. 주간 헌장 | 헌장 1개, 선택 도전 인장 0~2 | 서로 모순되는 인장 금지 |
| 4. 최종 요약 | 7밤 일정, 예상 적 역할, 출전·지원 목록 | 확정 뒤 Night 1 전까지만 수정 |

### 4.4 전투 사이 생활·정비 단계
- 전투 종료 후 모든 생존·전투 불능 상태는 주간 다음 밤 전에 회복된다.
- 생활 방문은 60~90초 이내의 짧은 장면이다.
- 플레이어는 방 하나를 선택해 해당 방·장식·몬스터 태그에 맞는 생활 장면을 본다.
- 본 적 없는 장면을 우선하되, 같은 주에 같은 장면은 재생하지 않는다.
- 장면은 건너뛸 수 있으며 건너뛰어도 해금 기록은 정상 저장된다.
- Night 4 뒤에는 전초기지 종류와 관계없이 출전 몬스터 1종을 벤치 후보와 교체할 수 있다.
- 교체는 성장 손실이나 유대 하락을 만들지 않는다.

### 4.5 패배·재도전
- 각 밤 시작 직전 안전 저장을 만든다.
- 정상 난이도에서는 패배 후 같은 밤을 재도전하거나 주간을 종료할 수 있다.
- 재도전 횟수는 주간 결과 카드에 기록하지만 엔딩 자체를 봉쇄하지 않는다.
- `seal_no_retry`를 선택한 주간만 패배 즉시 주간 종료로 처리한다.
- 주간 종료 시 프로필의 기존 캠페인·몬스터·엔딩 기록은 변하지 않는다.
- 부분 완료도 방문 도감과 본 생활 장면을 보존한다.

## 5. 규격화 편성과 주간 전투력
### 5.1 규격화 원칙
- 주간 모드는 장기 캠페인의 금화·마력·식량·레벨 누적치를 직접 사용하지 않는다.

- 자기 성 맵은 주간 적합성 검증을 통과한 저장 레이아웃 또는 승인된 기본 레이아웃을 사용한다.

## 7. 경쟁 마왕성 방문 도전
### 7.1 공통 구조

## 12. 마왕성 생활 기록과 제한적 방 꾸미기
### 12.1 설계 목표
- 몬스터가 전투 카드가 아니라 성의 거주자처럼 보이게 한다.
- 생활 장면은 유대와 서사를 보여 주지만 반복 클릭 보상을 만들지 않는다.
- 장식은 표현과 장면 조건만 제공하고 전투 능력치를 제공하지 않는다.
- 관리 화면을 생활 시뮬레이션으로 대체하지 않는다.

### 12.2 장식 앵커
- 각 방 데이터에 0~2개의 `decoration_anchor_id`를 둔다.
- 장식은 앵커 위치·레이어·최대 크기를 데이터로 검증한다.
- 장식 Sprite2D는 충돌·Navigation·RoomGraph에 참여하지 않는다.
- 전투 화면에서는 큰 장식을 반투명 또는 간소화할 수 있다.
- 주간 모드에서 배치한 장식은 프로필 프리셋에 저장하며 캠페인 맵 구조를 바꾸지 않는다.

## 13. 도감 기반 커스텀 도전 공방
### 13.1 해금과 목표
- 첫 `alliance_week` 클리어 뒤 제목 화면 또는 연대기에서 해금한다.
- 목적은 해금한 콘텐츠를 짧게 시험하는 것이며, 정규 캠페인을 대체하지 않는다.
- 한 도전은 단일 전투 5~12분 목표다.
- 도전 보상은 도장·장식·프리셋 슬롯뿐이다.

### 13.2 세 가지 진입 방식
| 방식 | 설명 |
| --- | --- |
| 공식 레시피 | 개발팀이 검증한 12개 고정 도전 |
| 안내형 제작 | 맵·지역·적 패키지·보스·제약을 제한 목록에서 선택 |
| 코드 불러오기 | 버전·데이터 해시·체크섬을 가진 오프라인 문자열 입력 |

### 13.3 안내형 제작 선택 항목
| 단계 | 선택 | 제한 |
| --- | --- | --- |
| 맵 | 자기 성 승인 레이아웃·게스트 성 3개·전초기지 5개 | 경로 검증 통과 맵만 |
| 환경 | 지역 규칙 0~1개 | Night 전용 스토리 규칙 제외 |

| 난수 | 자동 또는 사용자 seed | 32비트 정수 |

### 13.4 검증 규칙
- 입구에서 모든 필수 목표까지 경로가 존재한다.
- 보물 목표 적을 넣었으면 보물 또는 인장 목표가 존재한다.
- 시설 목표 적을 넣었으면 활성 시설이 하나 이상 존재한다.
- 보스는 최대 1명, named rival는 최대 1명이다.
- 동시 적 수 14명, 활성 장판 6개, 화면 효과 32개 상한을 지킨다.
- 환경 규칙과 제약 규칙을 합쳐 강제 이동·시야 방해 규칙은 최대 2개다.
- threat 예산은 선택 난이도 목표의 90~110%다.
- 동일 역할 하드 카운터 두 종이 동시에 전체 웨이브의 40%를 넘지 않는다.
- 주요 목표가 시작 8초 안에 즉시 파괴되는 조합은 거부한다.
- 코드의 데이터 해시가 현재 버전과 다르면 “호환 레시피로 변환” 또는 “읽기 전용 정보”만 제공한다.

## 14. 삼중 합동 공성
### 14.1 서사 위치
- Night 7의 고정 최종전이다.
- E16 `세 전선의 휴전문`과 세 인간 라이벌 관계가 높으면 “연합 심사전” 대사를 사용한다.
- 휴전 기록이 없으면 왕국·성광·길드가 마계 연맹 확대를 막기 위한 “긴급 합동 공성” 대사를 사용한다.
- 전투 규칙은 같고, 승패 대사와 후일담만 달라진다.
- 세 인물을 악당으로 만들지 않는다. 각자 책임과 방식 때문에 같은 작전표에 서는 것으로 묘사한다.

### 14.2 성능·가독성 원칙
- named rival는 동시에 최대 1명만 완전 활성 상태다.
- 다음 rival 진입 전 1.5~2.5초 전환 시간을 둔다.
- 이전 단계 일반 병력은 최대 3명만 남을 수 있다.
- 단계 전환 때 카메라 강제 이동 대신 상단 공지와 미니 초상으로 알린다.
- 각 단계의 고유 위험 색상과 아이콘을 분리한다.

### 14.3 Phase 1 — 로만의 합동 보급
- 시간 목표: 0~35초
- 로만은 예산 스택 3으로 시작한다.
- 보급 문서 상자 2개가 서로 다른 시설 경로에 등장한다.
- 상자가 목표에 도착하면 로만이 예산 스택 1을 얻고 주변 적을 회복한다.
- 상자를 파괴하거나 로만 HP를 55% 이하로 만들면 다음 단계 전환 조건이 열린다.
- 신규 기술 `coalition_supply_order`: 다음 단계 셀렌에게 잔여 예산만큼 보호막을 전달한다.
- 대응: 곱·코코 추격, 루미 표식, 미미 가짜 목표, 포포 경보.

### 14.4 Phase 2 — 셀렌의 봉인 행렬
- 시간 목표: 35~78초
- 셀렌은 전장에 봉인 표식 3개를 순서대로 설치한다.
- 표식이 완성되면 심장 또는 합동기 중 하나가 8초 정지한다.
- 표식 설치 중 셀렌에게 누적 피해 60 또는 스킬 중단을 주면 설치가 취소된다.
- 신규 기술 `sanctified_corridor`: 선택 방의 적 디버프를 정화하고 5초 아군 이동 속도를 낮춘다.
- 로만 예산이 남았으면 첫 봉인 표식에 보호막이 붙는다.
- 대응: 베베 중단·구조, 톡톡 방어 파괴, 루미/포포 경보, 곱 추격.

### 14.5 Phase 3 — 레온의 세 번째 맹세
- 시간 목표: 78~135초
- 레온은 3차의 적응형 자세 중 현재 주간 기여 지표에 맞는 하나를 선택한다.
- 로만·셀렌 단계에서 남긴 보급·봉인 결과가 레온 시작 보호막·진입 시간에 반영된다.
- 신규 기술 `allied_final_oath`: HP 45% 이하에서 1회, 이전 두 단계가 성공했으면 회복 12%와 상태 저항, 실패했으면 회복 6%만 적용한다.
- 최종 20% HP부터 왕좌 직선 경로를 우선하지만 도발·가짜 목표·봉쇄에 정상 반응한다.
- 레온의 돌진·외침·맹세 기존 약점은 유지한다.

### 14.6 경쟁 마왕 지원
| 선택 게스트 성 | Night 7 지원 | 제한 |
| --- | --- | --- |
| 브라사 | 선택 문턱 5초 철문·시설 보호막 50 | 공성·산성에 빨리 파괴 |
| 베스퍼 | 선택 몬스터 즉시 이동·3초 포획 면역 | 전투 불능·봉인 대상 불가 |
| 미렐라 | 선택 방 6초 회복·둔화 화단 | 정화에 지속시간 감소 |

### 14.7 최종전 실패·재도전
- Night 7 패배 시 Night 7 직전 안전 저장으로 돌아간다.
- 게스트 지원·보급 카드·레온 자세를 같은 조건으로 복원한다.
- 재도전 때 랜덤 도착 순서를 바꾸지 않는다.
- 도전 인장으로 재도전 금지를 선택한 경우 주간 결과 카드로 종료한다.

| 루미 | specialist 2~3명 | 위험 적 처치 시간 -20% | 후열 저격 시 생존 압박 |
| 미미 | 절도·인장 목표 | 실제 절도 성공 -60% | 목표 없는 웨이브 기여 감소 |
| 베베 | 장판·표적 분산 | 전투 불능 1명 감소 | 봉인 사슬 조합에서 구조량 -25% |
| 코코 | 엘리트 후열·보스 지원 | 표적 격퇴 -18% | 고방어 전열 시간 +10% |

### 15.4 합동기 상한
- 합동기 1회 총 피해는 같은 레벨 액티브 2개 합계의 120% 이하
- 합동기 보호막은 대상 최대 HP 30% 이하
- 합동기 회복은 대상 최대 HP 28% 이하
- 강제 이동 대상은 일반 적 최대 3명·boss 제외
- 스킬 중단은 boss에게 1회당 최대 0.8초
- 같은 전투에서 합동기 두 개의 효과가 겹칠 때 피해 감소·속도 버프는 기존 전역 상한 적용

### 15.5 주간 자동 플레이 프로필
| 프로필 | 편성 방향 | 검증 목적 |
| --- | --- | --- |

### 15.6 완주 목표
- 정상 난이도 자동 프로필 7개 중 6개 이상이 5개 seed에서 70% 이상 완주한다.
- 최소 조작 프로필도 최소 60% 완주한다.
- 왕관 집중 프로필이 가장 빠르더라도 잔여 전력·보조 목표에서 명확한 대가를 가진다.

## 18. UI/UX 설계
### 18.1 제목 화면과 모드 선택
| 모드 | 한 줄 설명 |
| --- | --- |
| 전선 연대기 | 레온·셀렌·로만과 30일 동안 맞서는 장기 회차 |
| 마계 의회 | 지역·전초기지·2층·경쟁 마왕을 운영하는 30일 회차 |
| 연맹 주간 | 7개 전투로 새 조합을 빠르게 시험하는 짧은 반복 모드 |
- 모드 카드에는 예상 시간, 핵심 선택 수, 최종 상대, 저장 상태를 표시한다.
- 진행 중 캠페인이 있으면 새 주간 시작 전에 별도 활성 슬롯 정책을 명확히 안내한다.
- 하나의 active_run만 유지하는 정책이면 현재 회차 종료·보관 여부를 확인한다.

### 18.2 주간 준비 화면
- Step 1 전력 편성: 5종 카드, 심장, 왕관, 합동기
- Step 2 연맹 노선: 게스트 성, 전초기지, 지역 카드 2개
- Step 3 헌장·인장: 이득과 대가를 한 줄씩 병렬 표시
- Step 4 7밤 일정: 각 밤의 맵·환경·보조 목표·예상 specialist 표시
- 선택 과정을 한 화면에 모두 펼치지 않는다.

### 18.3 주간 보드
- 7개의 밤 노드를 가로 타임라인으로 표시한다.
- 현재 밤, 완료 결과, 재도전 수, 다음 생활 방문을 표시한다.
- Night 3은 경쟁 마왕 문양, Night 4는 전초기지, Night 7은 세 인간 라이벌 초상을 사용한다.
- 보드에서 이전 결산과 생활 장면 기록을 다시 볼 수 있다.

### 18.4 생활 화면
- 중앙에 선택 방과 생활 애니메이션, 좌측에 방문 가능한 방, 우측에 장식 앵커와 장면 기록을 둔다.
- 장식을 드래그해 자유 좌표에 놓지 않는다. 앵커 선택 방식으로 배치한다.
- “전투 효과 없음” 문구를 장식 설명에 고정 표시한다.
- 장면 자동 넘김, 즉시 건너뛰기, 생활 애니메이션 줄이기 옵션을 제공한다.

### 18.5 도전 공방
- 좌측: 맵·환경·적·보스·제약 단계 탭
- 중앙: 선택 요약과 threat 게이지
- 우측: 검증 오류·경고·예상 역할
- 하단: 테스트 시작, 코드 복사, 코드 붙여넣기, 프리셋 저장
- 유효하지 않은 조합은 회색 버튼이 아니라 구체적 이유를 표시한다.

### 18.6 Night 7 HUD
- 상단에 `로만 보급 → 셀렌 봉인 → 레온 결투` 3단계 바를 표시한다.
- 현재 단계 위험과 다음 단계 진입 조건을 2줄 이내로 표시한다.
- 경쟁 마왕 지원 버튼은 심장·합동기 버튼과 색·위치를 분리한다.
- 봉인 상태일 때 어떤 시스템이 언제까지 정지하는지 표시한다.
- 보스 초상은 현재 활성 rival 하나만 크게 표시한다.

### 18.7 접근성
- 생활 애니메이션 자동 재생 끄기
- 전투 VFX 밀도 낮음·보통·높음

- outpost와 charter는 해금 목록에 존재한다.
- night_templates는 7개이고 각 템플릿의 map·wave·rule ID가 유효하다.
- safe_phase가 combat이면 저장을 거부한다.
- joint_siege_state는 Night 7에서만 존재한다.
- challenge code 원문은 active_run에 필수가 아니며 정규화 recipe만 저장한다.

- 같은 왕관 SFX의 최소 재생 간격은 0.12초다.
- 굴절·서리 효과음은 반복 시 음량과 피치를 작게 변형한다.

- 세 성의 맵과 목표가 초상 없이도 구분되는가

- 건너뛰기와 다시 보기가 충분히 빠른가
- 장식이 능력치 보상으로 오해되지 않는가
- 몬스터별 동작이 작은 화면에서도 구분되는가

### 25.5 도전 공방
- 유효한 레시피를 3분 안에 만들 수 있는가
- 오류 메시지가 수정 방향을 알려 주는가
- 공유 코드 입력이 Web과 PC에서 모두 가능한가
- 보상 없이도 조합 시험 자체가 재미있는가

### 25.6 삼중 공성
- 세 단계가 한 보스의 HP 세 줄이 아니라 다른 문제로 느껴지는가

## 28. 위험과 대응
### 위험 1. 모드가 세 개가 되어 제목 화면이 복잡해짐
예상 시간과 핵심 차이를 카드 한 줄로 표시하고 진행 중 저장은 별도 영역에 둔다.

### 위험 2. 주간 준비 선택이 다시 길어짐
4단계 화면으로 나누고 이전 주 편성 복사와 추천 편성을 제공한다.

### 위험 3. 규격화가 장기 육성을 무시하는 느낌
해금한 특화·진화·추억·왕관·합동기는 유지하고 raw level·경제만 정규화한다.

### 위험 4. 왕관 형태가 필수 정답
무왕관 헌장·보급 후보 확장·완주율 목표를 둔다.

### 위험 5. 게스트 성이 새 장르처럼 보임
모든 방문 콘텐츠를 방어전으로 유지하고 자유 탐험을 만들지 않는다.

### 위험 6. 생활 장면이 흐름을 끊음
90초 상한, 즉시 skip, unseen 우선, 자동 재생 끄기를 제공한다.

### 위험 7. 장식이 능력치 최적화로 변질
전투 수치 0을 코드 검사하고 설명에 고정 표시한다.

### 위험 8. 도전 공방이 자유 에디터로 팽창
선택 항목과 상한을 고정하고 raw JSON·스크립트 입력을 금지한다.

### 위험 9. 공유 코드 호환성
format version·data hash·checksum과 변환 안내를 둔다.

### 위험 10. 삼중 공성 화면 과밀
named rival 1명 활성, 단계 교대, 이전 병력 3명 상한을 둔다.

### 위험 11. 기존 13종 중 일부만 계속 선택
주간 결과에서 미사용 종 추천, 왕관 순환 도감, 역할별 공식 레시피를 제공하되 강제하지 않는다.

### 위험 12. 그래픽 제작량 폭증
신규 기본 종을 추가하지 않고 왕관 6종·적 4종·생활 13동작으로 상한을 고정한다.

### 위험 13. 최근 기록 저장 용량 증가
최근 10주만 상세 저장하고 누적 통계는 집계값만 남긴다.

### 위험 14. 신규 지역 규칙이 특정 빌드를 봉쇄
대칭 적용·고정 화로·방 이동·근접 대안 등 최소 두 가지 대응을 제공한다.

### 위험 15. 엔딩이 체크리스트 노가다
현재 주간 플레이 조건을 중심으로 하고 프로필 조건은 일부 숙련만 요구한다.
