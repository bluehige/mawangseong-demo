# 3차 업데이트 Phase 24 — 나머지 합동기와 합동 기억

## 구현 결과

- 합동기 카탈로그를 L01~L06 총 6개로 완성했다.
- L02 `유령 이사 대작전`: 최저 HP 아군을 푸딩 뒤로 이동하고, 푸딩이 4초 동안 피해 25%를 대신 받는다.
- L03 `달빛 냄새 추격`: 곱과 코코가 같은 표식 대상을 추격해 2회 타격하고 방어를 낮춘다.
- L04 `용융 갑각포`: 톡톡이 돌진한 지점에 충격 피해, 5초 화상, 방어 감소를 적용한다.
- 여섯 링크 모두 합동 기억 사건, 유대 45, 개인 추억 1개, 공동 출전 3일, 양쪽 역할 행동 조합 5회 조건을 사용한다.
- 공동 출전 일수와 역할 행동은 회차 데이터에 누적되며, 조건을 만족한 기억 사건만 관리 화면 후보가 된다.
- 합동 기억을 확인하면 프로필에 영구 해금하고 같은 사건은 다시 발생하지 않는다.
- 여섯 링크에 서로 구분되는 전용 VFX placeholder 식별자를 연결했다.
