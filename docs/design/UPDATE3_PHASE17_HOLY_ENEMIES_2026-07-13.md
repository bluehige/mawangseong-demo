# 3차 업데이트 Phase 17 완료 기록 — 나머지 성광 적

## 세션 17A — 성물 수호기사

- `reliquary_guard`와 `CHR_RELIQUARY_GUARD`, 성물 오라 데이터를 정규 카탈로그에 연결했다.
- 반경 155 안의 적군이 받는 마법 피해를 15%, 상태 이상 지속시간과 사기 피해를 25% 줄인다.
- 수호기사가 여러 명이어도 오라 수치는 중첩하지 않고 지속시간만 갱신한다.
- 수호기사 본인의 이동 속도는 기본 80에서 8% 감소한 73.6으로 실제 계산된다.
- 수호기사와 보호 대상에 오라 범위·상태 표시를 제공한다.
- 톡톡의 갑각 돌진은 수호기사 방어력 8을 6으로 낮추며, 오라 때문에 지속시간은 5초에서 3.75초로 줄어든다.

## 세션 17B — 성가 퇴마사

- `choir_exorcist`와 `CHR_CHOIR_EXORCIST`, 정화 성가 데이터를 정규 카탈로그에 연결했다.
- 살아 있는 심장실을 우선 목표로 삼되, 심장 목표 적은 동시에 한 명만 활성화한다. 현재 담당이 쓰러지면 대기 적 한 명이 승계한다.
- 심장실에 도달하면 방어 효과를 거친 실제 피해로 심장실을 공격하고, 심장실 비활성 뒤 왕좌로 복귀한다.
- 정화 성가는 1.2초 시전 뒤 반경 180 안 아군 최대 3명에게서 약화 효과를 하나씩 제거한다.
- 심장실 안에서 시전을 완료하면 기존 충전량은 줄이지 않고 4초 동안 석골·포식·몽등 심장의 신규 충전 획득을 모두 막는다.
- 4초가 지나면 충전이 자동으로 재개된다.
- 베베의 빗자루 소동으로 시전을 중단하면 정화와 충전 봉쇄가 발동하지 않는다.
- 월드 범위·남은 시전 시간과 HUD 충전 봉쇄 시간을 표시한다.
