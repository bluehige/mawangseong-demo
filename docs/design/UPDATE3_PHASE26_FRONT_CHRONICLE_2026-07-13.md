# 3차 업데이트 Phase 26 — 전선 연대기 완료 기록

## 완료 범위

- 전선 숙련 3종: 전선별 1·2·3회 클리어에서 34·68·100%
- 심장 숙련 3종과 세부 달성 기록 표시
- 레온·셀렌·로만 관계 수치와 실제 해금 기준 표시
- 합동 기억 6종의 해금 여부와 정확한 조건 표시
- 완료한 최근 회차를 최신 5개만 유지
- 전선별 후일담 카드 3종과 잠금 조건 표시
- 전투 능력치 대신 문패·기록 카드·대사 같은 이야기·꾸미기 보상만 사용
- 관리 화면의 `전선 연대기` 버튼과 저장 체크포인트 연결

## 저장 규칙

- 연대기 누적값은 저장 v4의 `profile`에 보존됩니다.
- 같은 회차를 다시 기록하면 중복 항목을 만들지 않고 기존 항목을 교체합니다.
- 여섯 번째 완료 회차가 들어오면 가장 오래된 요약 하나를 제거합니다.
- 저장을 읽을 때도 잘못 늘어난 과거 자료가 있으면 최신 5개로 정리합니다.
