# 3차 업데이트 Phase 3 완료 기록

## 결론

새 회차의 전선 선택·저장·복원 흐름을 추가했다. 기본 레온 `front_hero_oath`는 선택 가능하고, 셀렌·로만 전선은 프로필 조건과 첫 진입 초대장 결과에 따라 잠금 상태를 표시한다.

## 구현 범위

- `FrontCampaignService`: 해금 계산, 잠금 이유, 초대장, 선택 검증
- `FrontSelectionScreen`: 전선 3종 비교, 잠금 표시, 취소·복귀
- 엔딩 → 새 회차 전선 선택 → 계약 게시판 흐름
- 선택 대기·선택 완료 상태의 v1/v4 저장과 이어하기
- v4의 잘못된 front ID·잠긴 front 거부
- 1920×1080, 1366×768 화면 카드 배치 검증

## 의도적 제한

- 심장 선택과 `heart_chamber`는 아직 없다.
- 전선별 웨이브·보스 변경은 아직 적용하지 않는다.
- Phase 3에서 선택한 전선은 저장되지만, 실제 전투 구성은 기존 레온 캠페인과 동일하다.
