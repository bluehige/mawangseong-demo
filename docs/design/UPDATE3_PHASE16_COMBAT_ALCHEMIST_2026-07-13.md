# 3차 업데이트 Phase 16 완료 기록 — 전투 연금술사

## 완료 범위

- `combat_alchemist` 적과 `CHR_COMBAT_ALCHEMIST`, 산성 용액 기술 데이터를 정규 데이터에 연결했다.
- 0.8초 고정 위치 예고 뒤 반경 85의 산성 구역을 5초 동안 생성한다.
- 산성 구역 안 몬스터는 방어력 -2, 시설 수리 담당 몬스터는 수리량 -40%를 적용받는다.
- 같은 산성 효과는 중첩하지 않고 지속시간만 갱신하며, 겹친 구역의 지속 피해도 한 번만 계산한다.
- 산성 구역은 초당 2 피해를 주고, 전투 HUD와 월드 표시에서 위험 범위와 감소 수치를 확인할 수 있다.
- 베베는 예고 범위 밖으로 이동해 회피할 수 있고, 핀이 시전 중 연금술사를 처치하면 투척이 취소된다.
- 톡톡의 시설 수리는 산성 상태의 실제 수리 배율을 사용하며, 핵심 효율 감소 추정값이 20~35% 범위에 들도록 검증했다.
- 시설 목표 적 처리에서 전투 연금술사만 독립 경로를 사용하게 하여, 기존 레온 공성 자세와 공병의 시설 목표 배정을 보존했다.
- 전투 효과 정리는 컨테이너에서 즉시 분리한 뒤 삭제하도록 보완해 같은 프레임의 잔류 효과를 없앴다.
