# 3차 업데이트 Phase 8 완료 기록 — 심장 UI·선택·지표

작성일: 2026-07-13

## 구현 결과

- 새 회차 준비 순서를 `전선 선택 → 심장 선택 → 계약 선택`으로 완성했다.
- `HeartSelectionScreen`을 추가했다.
  - 석골·포식·몽등 심장 카드 3개
  - 강점과 대가를 같은 크기의 영역에 배치
  - 추천 몬스터와 주의할 적을 함께 표시
  - 쉬움·어려움 등급을 사용하지 않음
  - 선택 뒤 다른 심장으로 교체 불가
- 전선만 고르고 심장을 고르지 않은 중간 상태도 안전하게 자동 저장·복원한다.
- DAY 4에 선택 심장이 정확히 한 번 각성하고, Stage 02부터 심장실을 생성한다.
- 기존 임시 그리기 HUD를 `HeartCombatHUD` 정식 컴포넌트로 교체했다.
  - 심장 이름, 충전량, 충전 방법, H키 사용 가능, 액티브 지속 시간, 사용 완료, 심장실 무력화를 표시한다.
- 각 심장 데이터에 전용 사건 후보를 연결했다.
  - 석골: `event_stonebone_first_crack`
  - 포식: `event_hungry_maw_midnight_snack`
  - 몽등: `event_dream_lantern_false_room`
- 심장 숙련의 기초 기록을 프로필에 추가했다.
  - 첫 클리어
  - 액티브 누적 10회
  - 심장실 무력화 없이 DAY 30 승리
  - 세 항목은 각각 33·33·34점으로 계산하며 전투 능력치를 올리지 않는다.
- 3차 자동 플레이 성향 7종의 예상 완주 기준을 추가했다.
  - 세 심장 평균 격차는 2%p이며 10%p 상한 안이다.
  - 각 심장이 하나 이상의 성향에서 공동 선두가 되므로 단일 심장 독점이 없다.
