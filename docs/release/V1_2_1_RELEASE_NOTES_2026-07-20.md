# 제품 1.2.1 릴리스 노트

작성일: 2026-07-20

화면 표시: `1.2`

기술 SemVer: `1.2.1`

## 핵심 수정

- DAY 3 승리 결산을 저장·재실행한 뒤 성장 선택이 진행되지 않던 저장 ID 호환 문제를 수정했다.
- DAY 3 보스 안내 전에 패배한 뒤 재도전하면 전투 시작이 튜토리얼 단계에 막히던 문제를 수정했다.
- 자동전투 유닛의 경로점을 던전 안으로 보정하고, 일정 시간 이동하지 못하면 경로를 다시 찾도록 해 갇힘 현상을 복구한다.
- 안전하게 저장할 수 있는 화면 판정을 저장 시스템 한 곳으로 통합했다.

## 튜토리얼과 UI

- DAY 2 가시 복도에 실제 클릭 위치를 가리키는 큰 노란 표식과 클릭 배지를 추가했다.
- 가시 복도와 `[방 지침] → [함정 유도]` 설명을 짧고 실제 조작 순서와 일치하게 정리했다.
- 대화 초상화와 텍스트 패널 사이에 명확한 간격을 두어 의도하지 않은 겹침을 제거했다.
- 방 지침 용어를 실제 행동과 일치하도록 정리했다.

## 호환성과 배포

- 기존 1.2 저장 경로를 유지하며, 이전 저장을 그대로 이어서 플레이할 수 있다.
- 게임 화면에는 계속 `1.2`로 표시하고 프로젝트·Windows 파일·태그에는 `1.2.1`을 사용한다.
- Windows 패키지는 GitHub Release에서 내려받을 수 있다.
- PC Web: https://bluehige.github.io/mawangseong-web-playtest/
- 모바일 Web: https://bluehige.github.io/mawangseong-mobile-playtest/

## 검증 및 알려진 제한

- 기능 기준 SHA에서 전체 자동 검증 89/89와 Windows 실제 사용자 승리·패배 경로를 통과했다.
- 한글 이름 `마왕성`의 입력, 대화 표시, 저장 파일 보존과 재실행 이어하기를 확인했다.
- Windows 패키지는 코드 서명 인증서가 없어 서명되지 않았다.
- 자동화 입력 드라이버가 물리 한/영 키 코드를 제공하지 않아 Microsoft 한국어 IME의 물리 한/영 전환 자체는 수동 키보드 확인 항목으로 남는다.
