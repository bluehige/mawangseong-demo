# 제품 1.2 릴리스 노트

작성일: 2026-07-16

화면 표시: `1.2`

기술 SemVer: `1.2.0`

## 핵심 변경

- 단일 유닛 직접 이동·공격·수동 스킬 조작을 제거하고 전역·방 지시 중심의 자동 전투로 전환했다.
- 몬스터끼리의 몸체 충돌, 회피와 우회 이동을 제거했다. 공격·스킬 거리와 던전 경계 판정은 유지한다.
- 전투 속도 x1, x1.5, x2, x3와 일시정지를 PC·모바일에 제공한다.
- DAY 1~3 튜토리얼에서는 x1로 진행하고, 완료 뒤 첫 일반 전투에서 속도 기능을 한 번 소개한다.
- 초기 튜토리얼을 배치·방어·함정 유도·후퇴 지점과 자동 행동 관찰 중심으로 단순화했다.

## 입력과 UI 수정

- Windows 이름 입력창의 생명주기와 키 포커스를 수정해 IME 조합이 전역 단축키나 화면 재구성에 끊기지 않게 했다.
- 한글·숫자 혼합 이름의 편집, 길이 제한, 저장·불러오기 경로를 보강했다.
- 확대 글꼴에서 깨지던 상단 자원·DAY·체력·시설 효과와 하단 지시·속도 HUD 배열을 PC·모바일 기준으로 다시 배치했다.
- Web 빠른 시작 전에 유효한 회차 seed를 생성해 첫 자동 저장 경고가 노출되던 문제를 수정했다.

## 전투 밸런스

- 공격력·체력을 일괄 상향하지 않고 충돌 제거, 지시 AI와 x3 시간축으로 체감 전투 시간을 줄였다.
- DAY 1~3 대표 자동 전투는 모두 승리했고 x1 기준 36.0초, 35.6초, 68.7초였다. x3 체감 환산은 약 12.0초, 11.9초, 22.9초다.
- 방어 선택 간 가치는 절대 8초 기준 대신 현재 단축 전투에 맞는 최소 4초와 최속 선택 대비 15% 이상으로 검증한다.

## 호환성과 버전

- 구형 튜토리얼 진행 인덱스와 새 속도 안내 확인 필드를 정상화해 기존 저장을 이어갈 수 있다.
- 사용자 표기는 `1.2`, 프로젝트·Windows 파일·빌드 매니페스트는 `1.2.0`을 사용한다.
- 기존 `v0.*` 태그와 GitHub Release는 과거 공개 이력으로 보존하며 재지정하지 않는다.
