# 사용자가 직접 해야 하는 Steam 판매 준비

제가 대신 입력하거나 서명할 수 없는 항목만 모았다. 위에서부터 진행한다.

## 지금 바로

1. **계약 주체 결정**: 개인(sole proprietor) 또는 이미 설립한 법인 중 하나를
   고른다. 곧 법인을 만들 계획이면 가입 전에 먼저 결정한다.
2. **영문 법적 이름 일치 확인**: Steamworks 법적 이름, 세금 서류, 은행 계좌
   예금주가 정확히 같아야 한다. 브랜드명이나 닉네임을 법적 이름에 쓰지 않는다.
3. **자료 준비**: 신분·법인 자료, 영문 주소, 은행명·주소·SWIFT/계좌 정보,
   한국 납세자번호 등 세금 인터뷰에 필요한 자료를 준비한다.
4. **Steamworks 가입·서명**: NDA와 Steam Distribution Agreement를 읽고 직접
   전자서명한다.
5. **앱 등록비 결제**: Admin 권한 계정으로 게임 App Credit $100(지역 세금 별도)을
   결제하고 이 게임에 활성화한다. Steam Wallet 자금은 사용할 수 없다.
6. **세금·은행·신원 검증 완료**: 이메일로 추가 서류 요청이 오는지 확인한다.

## 한국 판매 전 별도 확인

Steamworks 가입과 별개로 아래 세 가지는 실제 계약 주체와 판매 구조에 따라 답이
달라지므로, 출시 국가에 대한민국을 포함하기 전에 각각 답변을 문서로 남긴다.

1. **사업자·세무**: 관할 세무서 또는 세무사에게 Steam 해외 플랫폼 정산을 받을
   개인/법인의 사업자등록, 업종 코드, 부가가치세·종합소득세/법인세와 외화 매출
   증빙 처리 방식을 확인한다. 국세청은 홈택스를 통한 사업자등록 신청과 서류
   전자제출을 안내한다: https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?cntntsId=7777&mi=2444
2. **게임 등급·게임사업자 신고**: Steam 콘텐츠 설문이 만드는 지역 등급만으로
   이 게임의 한국 유통 요건이 충족되는지, 게임제작업/배급업 신고 또는 별도
   등급분류가 필요한지 게임물관리위원회에 게임 내용과 Steam 배급 구조를 적어
   문의한다. 등급 신청·게임물관련사업자 안내: https://www.grac.or.kr/
3. **통신판매업**: Valve가 소비자 판매와 결제를 담당하는 현재 계약 구조에서
   개발자에게 통신판매업 신고 의무가 있는지 관할 시·군·구 또는 110에 확인한다.
   필요하다는 답변을 받은 경우 정부24 절차로 신고한다. 계약 구조를 보지 않고
   필요/불필요를 임의로 단정하지 않는다.

세무·등급·사업자 신고 완료 여부는 `steam/release_config.json`의
`legal_name_confirmed`와 `rights_audit_complete`를 `true`로 바꾸기 전에 확인한다.

가입 주소: https://partner.steamgames.com/steamdirect

## App ID가 생기면 저에게 줄 값

민감한 은행·세금·비밀번호 자료는 보내지 않는다. 아래 공개·운영 메타데이터만
알려주면 저장소의 0/placeholder를 제가 채울 수 있다.

- 기본 게임 App ID
- Windows Depot ID
- Demo App ID(만들었을 때)
- 스토어에 공개할 개발자명·퍼블리셔명
- 고객지원 이메일과 공개 웹사이트 또는 지원 페이지
- 한국어 공식 게임명 최종 승인
- 영문 게임명을 사용할지와 최종 표기
- 목표 출시 분기 또는 날짜
- 원하는 기준 가격과 사업 목표(모르면 함께 조사 가능)

## 출시 전 직접 확인·승인

- 게임명·로고 상표 검색과 필요한 권리 자문
- Steam 콘텐츠 설문의 폭력·공포·생성형 AI 답변
- 한국 판매의 게임 등급·사업·세무 의무
- 개인정보 처리방침의 사업자명, 이메일, 시행일
- 스토어 문구·이미지·가격·지원 언어가 실제 빌드와 일치하는지
- Valve 스토어와 빌드 피드백에 대한 최종 응답
- 출시 당일 Steamworks의 `Release App` → `Publish Now` → `Release Now`

## 계정 보안

SteamPipe 업로드는 개인 주계정이 아니라 `Edit App Metadata`와
`Publish App Changes To Steam`만 가진 전용 빌드 계정을 쓴다. 전화번호 또는 Steam
모바일 앱을 연결하고 Steam Guard를 유지한다. 비밀번호, Guard 코드, 세금·은행
자료, 인증서와 개인키를 저장소나 채팅에 올리지 않는다.
