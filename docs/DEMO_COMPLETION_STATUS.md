# 마왕성 데모 완료 상태

이 파일은 현재 데모가 어디까지 완성됐는지, 그리고 검수 후 무엇을 손보면 되는지 빠르게 확인하기 위해 필요하다.

작성일: 2026-07-01

## 1차 완료 결론

필수 플레이 루프는 1차 완료 상태다.

자동 스모크 테스트 기준으로 다음 흐름이 모두 통과했다. 스모크 테스트는 큰 기능이 최소한 끊기지 않는지 빠르게 확인하는 자동 점검이다.

```powershell
godot --headless --path . --scene res://tools/DemoSmokeTest.tscn
```

결과:

```text
DEMO_SMOKE_TEST: PASS
```

## 필수 체크리스트

- [x] 프로젝트 실행 시 관리 화면이 뜬다.
- [x] 방어 준비 후 전투 화면으로 이동한다.
- [x] 적이 입구에서 등장해 목표 방 또는 교전 목표를 잡는다.
- [x] 슬라임, 고블린, 임프가 자동으로 적을 공격한다.
- [x] 임프가 화염구 투사체를 발사한다.
- [x] 가시 복도 함정이 적에게 피해를 준다.
- [x] 전체 지침 변경이 전투 행동에 반영된다.
- [x] 방 지침 변경이 특정 방 행동에 반영된다.
- [x] 몬스터 하나를 직접 조종할 수 있다.
- [x] 왕좌의 방 HP가 0이면 패배한다.
- [x] 3일차 수련생 용사를 잡으면 데모 클리어가 뜬다.
- [x] 결산 화면이 표시된다.

## 검수 후 할 일

아래 항목은 1차 완성 뒤 실제 플레이 검수에서 손보면 좋다.

- 수동 플레이로 클릭 흐름, 전투 템포, 난이도 체감 확인
- UI 겹침, 텍스트 넘침, 작은 버튼 클릭감 조정
- 1~3일차 보상량, 적 체력, 왕좌 HP 피해량 밸런스 조정
- `GameRoot.gd`를 관리 화면, 전투 화면, HUD 단위로 분리
- 캐릭터 단일 이미지를 SpriteFrames 애니메이션으로 확장
- 미니맵, 사운드 이펙트, 튜토리얼 대사 추가
- 레벨업 스킬 후보 선택 UI 추가
- 자유 건설 시스템 확장

## 현재 기준

이번 완료 기준은 "상용 출시 수준"이 아니라 "처음부터 3일차 클리어 또는 패배까지 연결되는 플레이 가능한 데모"다.
