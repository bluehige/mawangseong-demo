## Prompt Set Used

Shared prompt frame:

```text
Use case: stylized-concept
Asset type: 512x512 game dialogue portrait emotion variant
Primary request: Create an emotion variant of trainee hero Leon with the emotion tag "<emotion>".
Input image role: the previously shown local portrait_hero_leon.png is the identity reference; preserve the same character identity.
Subject: trainee hero, sincere but inexperienced, bright armor, rulebook/manual habit; clean heroic armor outline, short cape or shoulder guard, earnest expression.
Style/medium: dark fantasy comedy, polished painterly 2D game dialogue portrait.
Composition/framing: bust portrait, three-quarter view, centered, readable armor silhouette, generous padding, square portrait.
Color palette: blue, white metal, warm gold trim, deep dungeon backdrop.
Constraints: preserve the base character's face, blond hair, blue eyes, costume proportions, armor silhouette, palette, and youthful trainee feel; change only expression, head tilt, and subtle shoulder/hand acting; no labels, no text, no watermark.
Avoid: seasoned veteran, dark knight, villain face, extra characters, gore, text, watermark.
```

Emotion-specific mood lines:

- `heroic`: brave entrance energy, upright posture, confident but still trainee, bright determined eyes.
- `flustered`: embarrassed panic, widened blue eyes, nervous blush, mouth caught mid-correction, small bead of sweat.
- `manual`: slightly over-serious while checking a hero manual or rulebook; book cue allowed only if it does not cover the face and has no readable text.
- `determined`: serious resolve, steady forward gaze, tense final-push feeling without comedy exaggeration.
- `defeated`: tired and defeated but not injured, lowered shoulders, humbled expression, quiet loss after battle.
