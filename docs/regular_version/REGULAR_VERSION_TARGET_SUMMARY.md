# 정규버전 1.0 목표 요약

작성일: 2026-07-02

## 1.0 범위

정규버전 1.0은 대형 정식판이 아니라, 완주 가능한 소규모 정규판으로 잡는다.

- 캠페인: 30일
- 챕터: 5개
- 플레이 시간: 약 4~6시간
- 마왕성 등급: F급 동굴에서 A급 마왕성까지
- 몬스터: 일반 12종, 희귀 4종
- 적: 일반 침입자 12종, 보스 5명
- 방: 20~24종
- 함정: 10~12종
- 아이템: 60~80종
- 이벤트: 100~120개
- 엔딩: 3~5개
- 난이도: 쉬움, 보통, 어려움

## 포함할 핵심 시스템

- 쿼터뷰 소켓 연결식 모듈 던전
- 바닥 셀 기반 이동 가능 영역
- 실시간 자동 전투
- 전체 지침, 방 지침, 직접 조종
- 몬스터 레벨, 스킬 슬롯, 진화 1단계
- 인간 악명, 마계 명성
- 전리품, 재료, 설계도, 유물, 소모품
- 메인 이벤트, 몬스터 이벤트, 랜덤 이벤트, 보스 이벤트
- 저장/불러오기
- 몬스터, 적, 아이템, 일부 이벤트 도감

## 1.0에서 미루는 것

- 완전 자유건설
- 다층 마왕성 전체 구현
- 몬스터 진화 2~3단계
- 장비 세트 시스템
- 대규모 전쟁 모드
- 온라인 요소

## 제작 원칙

웹GPT가 기획 문서를 만들고, Codex가 그 문서를 JSON과 Godot 코드로 흡수한다.

따라서 신규 콘텐츠는 코드에 직접 박지 않는다. 몬스터, 적, 방, 스킬, 웨이브, 아이템, 이벤트, 밸런스 값은 JSON 또는 CSV 기반으로 관리한다.
