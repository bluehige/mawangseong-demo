## 완료 범위

DAY 22부터 DAY 27까지의 캠페인, 전투 기믹, 실제 스킬 동작, 밸런스 시뮬레이션과 네 번째 마왕성 진화를 연결했다. DAY 27에서 승리하기 전에는 Stage 03을 유지하고, 성채 심장 쟁탈전에서 승리한 뒤에만 Stage 04 `대마왕성`으로 진화한다.

| 날짜 | 이야기와 플레이 변화 |
|---|---|
| DAY 22 | 감시탑의 두 번째 봉화. 감시초소 사각을 기록하려는 조사대를 막는다. |
| DAY 23 | 두 번째 승급 슬롯을 열고, 서로 다른 역할의 승급 몬스터 둘을 운용할 수 있게 한다. |
| DAY 24 | 레온이 새로 그린 마왕성 지도를 회수하고 다음 날 재도전을 예고한다. |
| DAY 25 | 기존 수련생 용사 레온이 재등장해 실제 `돌진`과 `용기의 외침`을 사용한다. |
| DAY 26 | 왕국이 마왕성을 정식 위협으로 등록한다. 셀렌의 진군 지휘와 공병 시설 교란이 함께 작동한다. |
| DAY 27 | 공병대의 성채 심장 쟁탈전을 막으면 최종 진화 조건이 열리고 Stage 04가 즉시 적용된다. |

이번 구간에는 신규 캐릭터를 추가하지 않았다. 레온과 셀렌은 이미 제작된 캐릭터와 애니메이션을 재사용했고, 자세만 바꾸는 대체 연출은 허용하지 않았다.

## 실제 애니메이션과 기술

- 레온은 `idle 2 / move 4 / attack 4 / skill 4 / down 2`의 필수 전투 애니메이션을 모두 사용한다.
- 레온의 `돌진`은 0.32초 동안 목표 방향으로 실제 120px를 이동한다. 이동 중에는 일반 추적 AI가 경로를 덮어쓰지 못하며, 이동이 끝난 뒤 도착 지점 반경 72 안에만 피해 20을 준다.
- `용기의 외침`은 5초 동안 반경 360 안의 왕국군 이동 속도를 12% 높이고 공격 간격을 18% 줄인다. 재사용 대기시간은 10초다.
- DAY 26 셀렌의 `진군 지휘`는 DAY 21과 같은 실제 강화, 기술 모션, 오라, 격퇴 시 즉시 해제를 모두 수행한다.
- 공병은 먼저 나온 공병이 이미 격퇴됐더라도 전투 중 목표 이력을 기억해 서로 다른 시설을 노린다. 도착 후에는 전용 기술 모션으로 해당 시설만 무력화한다.
- 자동 검사는 각 기술의 피해·이동·강화·만료·결산 수치까지 확인한다. 단순히 스프라이트 슬롯만 존재하는지는 통과 조건이 아니다.

## Stage 04 대마왕성 변화

- 구역 수: `9 → 11`
- 신규 구역: `최정예 주둔지`, `서부 건설 구역`
- 왕좌 체력: `2100 → 2500`
- 기존 시설: 단계별 내구도와 배치 정원 증가
- 보조 시설: 수호핵, 감시초소, 회복 시설 등의 단계 배율 강화
- 다중 시설: 같은 종류의 방이 둘 이상이어도 각각 독립적으로 효과와 무력화 상태를 계산
- 최정예 주둔지: 기본 병영이 무력화되어도 해당 방에 배치된 몬스터에게 별도 병영 효과 적용
- 상태 표시: 같은 역할의 시설이 여러 개면 `병영(작동 1/2)`처럼 부분 작동 상태를 표시
- 실제 수치 표시: Stage 04 회복 시설은 `+12.0/s`, 감시초소는 실제 판정과 같은 2단계 인접 구역까지 강조 표시
- 전투 HUD: 병영·감시초소·회복 시설·마력 수호핵 네 효과를 모두 표시하며, 네 번째 수호핵 줄이 잘리지 않도록 패널 높이를 내용에 맞춰 확장
- 시설 변경 창: 진화 전 기본값이 아니라 Stage 04 실제 체력·정원을 표시한다. 병영 `770/7`, 보물고 `570/5`, 회복 `670/5`, 감시 `700/6`, 수호핵 `740/4`를 실제 화면에서 확인했다. 빈 슬롯은 진화 보너스를 더하지 않고 `체력 200 / 배치 불가`로 구분한다.
- 진화 결산: `방어 성공 · 대마왕성 완성` 전용 제목, 최종 진화 배너와 강조 애니메이션 표시
- DAY 28로 넘어가도 11구역, 2500 왕좌 체력, Stage 04 진화 이력 유지
