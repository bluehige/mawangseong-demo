## 참가자 사후 응답

각 점수는 1점이 가장 낮고 5점이 가장 높다.

- 재미: 1 / 2 / 3 / 4 / 5
- 무엇을 해야 하는지 이해됨: 1 / 2 / 3 / 4 / 5
- 피로함: 1 / 2 / 3 / 4 / 5
- 가장 재미있었던 결정은 무엇이었나요?
- 가장 이해하기 어려웠던 화면이나 규칙은 무엇이었나요?
- 한 번 더 플레이한다면 무엇을 다르게 하겠나요?
- 이 게임을 친구에게 한 문장으로 어떻게 설명하겠나요?

주관 점수와 자유 응답은 자동 PASS를 만들지 않는다. 진행자는 참가자의 표현을 고쳐 쓰지 않는다.
