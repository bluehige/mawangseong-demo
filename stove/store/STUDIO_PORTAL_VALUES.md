# STOVE Studio 포털 입력값

작성 기준일: 2026-07-15

## 생성 전에 승인할 ID

프로젝트 ID와 게임 ID는 생성 뒤 수정이 어렵거나 불가능하므로 아래 값이 마음에
들지 않으면 **생성 전에** 바꾼다.

| 항목 | 추천값 | 상태 |
|---|---|---|
| 그룹명 | `REPLACE_WITH_GROUP_NAME` | 공개 개발자/퍼블리셔명 결정 대기 |
| 그룹 ID | `REPLACE_WITH_GROUP_ID` | 영문 소문자 브랜드 ID 결정 대기 |
| 프로젝트 타입 | `PC` | 확정 |
| 프로젝트명 | `마왕님, 마왕성은 누가 지켜요?` | 한국어 공식명 승인 대기 |
| 프로젝트 ID | `who-guards-demon-castle` | 추천안 |
| 상품 타입 | `BASIC` | 유료 정식판 |
| 상품명 | `마왕님, 마왕성은 누가 지켜요?` | 한국어 공식명 승인 대기 |
| 게임 ID | `who-guards-demon-castle` | 추천안 |

그룹 ID에는 주민등록상 이름이나 개인 이메일을 넣지 않는다. 프로젝트/게임 ID는
표시명이 아니라 내부 식별자이므로 영문 소문자와 하이픈을 사용한다.

## 기본 상품 설정

| 항목 | 값 |
|---|---|
| 국가/지역 | 대한민국 우선, Steam 동시 출시 국가와 추후 대조 |
| 지원 OS | Windows |
| 아키텍처 | 64-bit x86_64 |
| 지원 언어 | 한국어 인터페이스·자막, 음성 없음 |
| 영어 지원 | 게임 내 영문화 완료 전 체크하지 않음 |
| 싱글 플레이 | 예 |
| 멀티플레이 | 아니오 |
| 업적 | 출시 시점 미지원 |
| 인앱 결제 | 없음 |
| DLC | 출시 시점 없음 |
| 고객지원 | `REPLACE_WITH_SUPPORT_PAGE` |
| 문의 이메일 | `REPLACE_WITH_SUPPORT_EMAIL` |

## 실행 설정 초안 — 사운드 확정 뒤 입력

| 항목 | 값 |
|---|---|
| 실행 파일 | `MawangCastle.exe` |
| 실행 인자 | 없음 |
| 작업 폴더 | 패키지 루트 |
| 관리자 권한 | 불필요 |
| 별도 런처 | 없음 |

최종 Windows 패키지의 실제 파일명과 STOVE Uploader 결과를 확인하기 전에는 실행
설정을 제출하지 않는다.

## 공개 이름

- 한국어 공식명: `마왕님, 마왕성은 누가 지켜요?`
- 영문명 초안: `Who Guards the Demon Castle?`
- 개발자명: `REPLACE_WITH_DEVELOPER_NAME`
- 퍼블리셔명: `REPLACE_WITH_PUBLISHER_NAME`

영문명은 게임 내 영문판과 상표 검토가 끝나기 전까지 초안이다.
