# 자체등급분류용 게임 설명서

작성 기준일: 2026-07-15
게임명: **마왕님, 마왕성은 누가 지켜요?**
플랫폼: Windows PC
장르: 2D 쿼터뷰 마왕성 건설·방어 시뮬레이션
플레이 형태: 싱글 플레이
심의 빌드 버전: `v2.0.1`

## 1. 게임 개요

플레이어는 F급 신입 마왕이 되어 작은 동굴을 마왕성으로 성장시킨다. 방과 시설을
배치하고 슬라임, 고블린, 임프 등 몬스터를 성장·지휘해 인간 왕국의 침입을
막는다. 한 회차는 DAY 1부터 DAY 30까지 관리, 전투, 원정, 이벤트와 결말로
진행된다. 선택한 전선, 계약, 동료 관계, 왕관과 마왕성 심장에 따라 전투와 엔딩이
달라지고 일부 기록은 다음 회차에 남는다.

현금 결제, 유저 간 거래, 온라인 상호작용, 카지노·베팅·포커·룰렛 등 도박 모사
기능은 없다.

## 2. 주요 플레이 흐름

1. 이름 등록과 튜토리얼을 진행한다.
2. 관리 화면에서 방·시설을 배치하고 몬스터를 배정·성장시킨다.
3. 전투 화면에서 자동 전투를 관찰하거나 몬스터를 선택해 이동·공격·기술을
   직접 지시한다.
4. 전투 결과에 따라 자원, 관계, 연대기와 다음 DAY의 선택지가 갱신된다.
5. 지역·전초기지·전선을 고르고 DAY 30 결전을 진행한다.
6. 조건에 따른 엔딩과 후일담을 확인하고 다음 회차를 시작한다.

| 인물 | 역할과 내용 |
|---|---|
| 신입 마왕 | 플레이어가 이름을 정하는 주인공. 마왕성을 건설하고 몬스터를 지휘한다. |
| 바티 | 초반 안내와 마왕 등록 절차를 돕는 조력자. |
| 골딘 | 자원과 관리 업무를 안내하는 회계 담당자. |
| 푸딩 | 방어와 보호에 강한 슬라임 동료. |
| 곱 | 근접 전투를 담당하는 고블린 동료. |
| 핀 | 화염 마법을 사용하는 임프 동료. |
| 모리·돌콩·두둠·루미·미미·베베·코코·톡톡 | 계약과 성장으로 합류하는 몬스터 동료들. |
| 레온 | 성장하며 여러 차례 마왕성에 도전하는 인간 용사. |
| 셀렌 | 성기사 진영의 지휘관. 정화와 방어 기술을 사용한다. |
| 로만 | 길드 조사와 보급·감사를 지휘하는 경쟁 인물. |
| 에블린 | 왕국의 최종 전략을 설계하는 인물. |

## 4. 조작법

| 입력 | 동작 |
|---|---|
| 마우스 왼쪽 클릭 | UI 선택, 방·유닛 선택, 이동/공격 지점 지정 |
| 마우스 오른쪽 클릭 | 선택 취소 또는 상황별 보조 명령 |
| 마우스 휠 | 전투 화면 확대/축소 |
| Space | 전투 일시정지/재개, 대화 진행 |
| Enter | 대화 진행 |
| Tab | 다음 몬스터 선택 |
| 1 / 2 / 3 | 선택한 몬스터의 기술 사용 |
| H | 마왕성 심장 능력 사용 |
| J / K | 장착한 합동기 사용 |
| Q / E | 1층/2층 전환 |
| Esc | 설정·몬스터 화면 닫기, 관리 동작 취소 |

출시 빌드에서 디버그 전용 단축키는 사용자 설명과 심의 영상에 포함하지 않는다.

## 5. 등급 관련 표현

- 판타지 폭력: 인간형 침입자와 몬스터가 무기, 마법, 함정으로 서로 공격한다.
- 표현 방식: 2D 캐릭터 애니메이션, 타격 효과와 체력 감소. 사실적인 신체 훼손,
  절단, 장기 노출, 선혈 묘사는 관찰되지 않았다.
- 공포: 악마, 해골, 던전, 유령, 마왕성 소재와 일부 공포 관련 대사가 있으나
  전반적으로 귀엽고 희화화된 판타지 표현이다.
- 선정성·노출: 관찰된 요소 없음.
- 욕설·차별 표현: 현재 텍스트 검색에서 관찰된 요소 없음.
- 음주·흡연·마약: 관찰된 요소 없음.
- 범죄 조장: 침입, 보급 방해 등 판타지 전쟁 서사가 있으나 현실 범죄의 상세한
  실행법이나 미화는 관찰되지 않았다.
- 사행성: 현금 결제, 유료 확률형 아이템, 유저 간 거래, 베팅·카지노 모사 없음.

최종 제출 전 출시 후보 빌드 전체 플레이와 모든 CG·언어 파일을 다시 확인한다.

## 7. 치트·심의용 접근법

- 일반 사용자용 치트 코드: 없음
- 심의용 빌드 우회 기능: 판매 빌드에는 별도 치트나 우회 기능이 없다.
- 엔딩 확인 방법: 등록된 전체 엔딩 23개를 `04_all_endings_v2.0.1.mp4`에
  카탈로그 코드·엔딩 ID·제목과 함께 순서대로 수록한다. 자동 캡처 도구는 개발
  저장소에서만 사용하며 사용자 판매 빌드에는 포함하거나 노출하지 않는다.

## 8. 개인정보와 네트워크

게임은 개발자 서버, 광고 SDK, 분석 SDK, 계정 서버를 사용하지 않는다. 진행도,
설정과 커스텀 맵은 로컬에 저장된다. 플레이어 이름은 로컬 저장에만 사용되며
자동 전송되지 않는다. STOVE의 플랫폼 기능을 추가할 경우 이 항목을 다시 갱신한다.
