# 자체등급분류 영상 촬영표

상태: **v2.0.1 심의용 영상 4종 생성·검증 완료 / 포털 제출 전 빌드 교체 필요**

## 공통 촬영 조건

- 최종 제출 대상과 같은 Windows 빌드·버전·언어를 사용한다.
- 게임명과 버전을 영상 시작 부분에 표시하거나 파일명/설명에 기록한다.
- 편집으로 등급 요소를 누락하지 않는다. 로딩 구간만 최소한으로 줄인다.
- 게임 소리와 화면이 실제 출시 상태와 일치하도록 녹화한다.
- 각 영상의 시작·종료, DAY, 전투, 등급 요소와 엔딩 ID를 타임코드 표에 기록한다.

던전 소재를 포함하고 `RATING_ELEMENTS_TIMECODES.txt`에 위치를 기록했다.

## 엔딩 대조표

최종 출시 데이터의 23개 엔딩을 자동 순회했다. 캡처 중 요청 ID와 실제 화면 데이터의
ID가 다르면 즉시 실패하며, 이번 실행은 E00~E22 23/23을 통과했다. 각 엔딩은 영상
좌상단 표기로 직접 대조할 수 있고, CG 원본은 제출 패키지의
`CG_IMAGES/ENDINGS_INDIVIDUAL/`에도 별도 수록했다.
