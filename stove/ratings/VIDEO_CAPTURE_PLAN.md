# 자체등급분류 영상 촬영표

상태: **사운드 추가와 출시 후보 빌드 확정 전 촬영 금지**

## 공통 촬영 조건

- 최종 제출 대상과 같은 Windows 빌드·버전·언어를 사용한다.
- 게임명과 버전을 영상 시작 부분에 표시하거나 파일명/설명에 기록한다.
- 편집으로 등급 요소를 누락하지 않는다. 로딩 구간만 최소한으로 줄인다.
- 게임 소리와 화면이 실제 출시 상태와 일치하도록 녹화한다.
- 각 영상의 시작·종료, DAY, 전투, 등급 요소와 엔딩 ID를 타임코드 표에 기록한다.

## 필수 영상

| 파일명 | 최소 길이 | 포함할 내용 |
|---|---:|---|
| `01_early_game.mp4` | 7분 | 이름 등록, 튜토리얼, 초기 관리, 첫 방 배치, 첫 전투 |
| `02_mid_game.mp4` | 7분 | 성장·승급·특화, 시설 운영, 중반 원정과 전투, 합동기 |
| `03_late_game.mp4` | 7분 | 상층/전초기지, 후반 보스, 강한 폭력·공포 표현, DAY 30 흐름 |
| `04_all_endings.mp4` 또는 분할본 | 전체 | 모든 출시 엔딩과 후일담. 엔딩 ID별 타임코드 포함 |
| `05_rating_elements.mp4` | 충분한 길이 | 무기·마법·함정 전투, 피격·전투 불능, 해골·유령·악마 등 공포 소재 |

## 엔딩 대조표

최종 출시 데이터에서 엔딩 ID 목록을 다시 추출해 아래 표를 완성한다.

| 엔딩 ID | 제목 | 영상 파일 | 시작 타임코드 | 확인 |
|---|---|---|---|---|
| `REPLACE_AFTER_FINAL_BUILD` |  |  |  | 대기 |

## 제출 전 검수

- [ ] 초·중·후반 각각 실제 플레이 7분 이상
- [ ] 다중 엔딩 전부 포함
- [ ] 폭력·공포가 가장 강한 장면 포함
- [ ] 선정성·약물·사행성 없음 판단을 모든 CG·텍스트와 대조
- [ ] 영상 속 빌드와 업로드한 심의 빌드가 동일
- [ ] 파일 또는 다운로드 링크 권한을 외부 검수자가 열 수 있음
- [ ] 주민등록정보, 계정 이메일, 개인 폴더명 등 개인정보 노출 없음
