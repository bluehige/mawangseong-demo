# STOVE 자체등급 설문 답변 초안

작성 기준일: 2026-07-15
상태: **v2.0.1 자료 준비 완료 / STOVE 포털의 v0.2.1 빌드를 v2.0.1로 교체하기 전 제출 금지**

현재 게임은 비현실적·희화화된 판타지 전투와 경미한 공포 소재가 핵심이다. 보수적
내부 예상은 전체이용가 또는 12세이용가 범위이지만, 최종 등급은 STOVE 심사 결과로
결정되며 이 문서가 등급을 보장하지 않는다.

## 일곱 가지 등급 요소

| 요소 | 초안 판단 | 근거와 제출 설명 | 최종 확인 |
|---|---|---|---|
| 폭력성 | 있음, 비현실적 판타지 | 2D 몬스터와 인간형 침입자가 무기·마법·함정으로 전투. 체력 감소와 타격 효과가 있으나 사실적 선혈·절단·신체 훼손 없음 | 모든 전투·엔딩 영상 확인 |
| 선정성 | 없음 | 성행위, 성적 대사, 노출 중심 의상·연출을 관찰하지 못함 | 모든 캐릭터·CG 확인 |
| 공포 | 경미 | 악마·해골·유령·던전·마왕성 소재. 귀엽고 코믹한 2D 표현이 중심 | 후반 엔딩 CG 확인 |
| 언어 | 부적절 표현 없음 | 현재 데이터·코드 검색에서 욕설과 차별적 비속어를 관찰하지 못함 | 생성된 전체 언어 파일 재검색 |
| 약물 | 없음 | 술·담배·마약 사용 또는 조장 장면을 관찰하지 못함. 판타지 회복·마력은 약물 묘사 아님 | 아이템·이벤트 전체 확인 |
| 범죄 | 경미한 판타지 갈등 | 침입·약탈·보급 방해 같은 판타지 전쟁 서사가 있으나 현실 범죄 실행법·미화 없음 | 원정·이벤트 문구 확인 |
| 사행성 | 없음 | 현금 결제, 유료 확률형 아이템, 유저 간 거래, 베팅, 룰렛, 카드 도박, 카지노 없음 | 최종 경제 시스템 확인 |

## 심사 화면 입력 초안

- 제작 국가: 대한민국
- 한 줄 설명: `방과 시설을 배치하고 몬스터를 성장·지휘해 30일 동안 인간 왕국의 침입을 막는 2D 쿼터뷰 마왕성 방어 시뮬레이션 게임.`
- 온라인 상호작용: 없음
- 인앱 결제: 없음
- 확률형 유료 아이템: 없음
- 사용자 제작 콘텐츠: 로컬 커스텀 맵만 존재하며 온라인 공유 기능 없음
- 개인정보 전송: 개발자 서버로 전송 없음
