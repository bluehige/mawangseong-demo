# 05. QA / 밸런스 체크리스트

이 문서는 데모 구현 후 기능 검수와 밸런스 조정에 사용한다.

---

## 1. 실행 체크

| 체크 | 항목 | 기준 |
|---|---|---|
| ☐ | 프로젝트 실행 | 오류 없이 메인 화면 표시 |
| ☐ | 관리 화면 진입 | 마왕성 관리 UI 표시 |
| ☐ | 전투 화면 진입 | 방어 준비 후 전투 시작 |
| ☐ | 결산 화면 | 전투 종료 후 결과 표시 |
| ☐ | 다음 날 진행 | 1일차 → 2일차 → 3일차 진행 |
| ☐ | 클리어 | 3일차 견습 용사 격퇴 시 클리어 표시 |
| ☐ | 패배 | 왕좌의 방 HP 0 시 패배 표시 |

---

## 2. 맵/방 체크

| 체크 | 항목 | 기준 |
|---|---|---|
| ☐ | 입구 | 적 등장 위치가 명확함 |
| ☐ | 가시 복도 | 함정 위치가 눈에 보임 |
| ☐ | 병영 | 몬스터 대기 방으로 표시됨 |
| ☐ | 보물 보관실 | 도적 목표로 작동 |
| ☐ | 회복 둥지 | 후퇴 목표로 작동 |
| ☐ | 왕좌의 방 | 핵심 방어 목표로 작동 |
| ☐ | 방 선택 | 클릭 시 정보 표시 |
| ☐ | 경로 | 입구에서 목표 방까지 이동 가능 |
| ☐ | 쿼터뷰 가독성 | 방의 바닥/벽/출입구/소품이 3/4 하향 시점으로 읽힘 |
| ☐ | 가림 처리 | 벽/기둥/소품이 유닛과 핵심 UI를 과도하게 가리지 않음 |

---

## 3. 유닛 체크

| 체크 | 항목 | 기준 |
|---|---|---|
| ☐ | 슬라임 생성 | 입구/가시 복도 근처 배치 가능 |
| ☐ | 고블린 생성 | 근접 공격 가능 |
| ☐ | 임프 생성 | 원거리 공격 가능 |
| ☐ | 탐험가 생성 | 왕좌 방향으로 이동 |
| ☐ | 도적 생성 | 보물 보관실로 이동 |
| ☐ | 견습 용사 생성 | 왕좌의 방으로 이동 |
| ☐ | HP 표시 | 유닛 HP가 눈에 보임 |
| ☐ | 전투불능 | HP 0에서 DOWN 처리 |
| ☐ | 발밑 피벗 | 선택 링, HP 바, 충돌체, 이동 목표가 발밑 기준으로 맞음 |
| ☐ | 앞뒤 정렬 | 화면 y값 기준으로 앞 유닛이 뒤 유닛보다 앞에 보임 |

---

## 4. 전투 체크

| 체크 | 항목 | 기준 |
|---|---|---|
| ☐ | 근접 공격 | 사거리 내 공격 발생 |
| ☐ | 원거리 공격 | 임프 화염구 발사 |
| ☐ | 투사체 명중 | 화염구가 목표에 피해 |
| ☐ | 피해 공식 | 방어력에 따른 피해 감소 적용 |
| ☐ | 타깃 탐색 | 가장 적절한 적을 공격 |
| ☐ | 도적 약탈 | 보물고 도착 시 약탈 게이지 또는 손실 발생 |
| ☐ | 견습 용사 압박 | 3일차에 왕좌를 위협함 |
| ☐ | 가시 함정 | 적 진입 시 피해/둔화 |

## 5. 행동지침 체크

### 전체 지침

| 체크 | 지침 | 기준 |
|---|---|---|
| ☐ | 사수 | 몬스터가 방을 벗어나 과도하게 추격하지 않음 |
| ☐ | 총공격 | 몬스터가 더 적극적으로 적을 추격 |
| ☐ | 생존 우선 | HP 35% 이하 후퇴 |

### 방 지침

| 체크 | 지침 | 기준 |
|---|---|---|
| ☐ | 입구 봉쇄 | 슬라임/근접 몬스터가 입구 쪽을 막음 |
| ☐ | 함정 유도 | 적이 가시 복도 쪽으로 유도됨 |
| ☐ | 후퇴선 유지 | 약한 몬스터가 회복 둥지 쪽으로 빠짐 |

## 10. 전투 감각 체크

| 질문 | 좋은 상태 | 나쁜 상태 |
|---|---|---|
| 전투가 읽히는가? | 누가 누구와 싸우는지 보임 | 유닛이 뭉쳐서 안 보임 |
| 쿼터뷰가 기능을 방해하지 않는가? | 깊이감은 있지만 선택/이동/충돌 기준이 명확함 | 장식이나 벽 때문에 유닛 위치가 헷갈림 |
| 지침이 의미 있는가? | 명령 변경 후 행동이 달라짐 | 버튼을 눌러도 차이가 없음 |
| 직접 조종이 필요한가? | 위기 순간 직접 조종이 유리함 | 자동 전투만 봐도 충분함 |
| 패배 이유가 보이는가? | 도적 약탈/왕좌 돌파 등 원인이 명확함 | 갑자기 졌다고 느껴짐 |
| 데모가 짧고 명확한가? | 10~15분 내 끝남 | 기능 설명이 길고 지루함 |

## 11. 버그 우선순위

| 등급 | 기준 | 예시 |
|---|---|---|
| P0 | 진행 불가 | 전투 시작 불가, 게임 크래시 |
| P1 | 핵심 기능 실패 | 적이 이동하지 않음, 공격 안 됨 |
| P2 | 주요 불편 | UI 갱신 안 됨, 지침 상태 표시 오류 |
| P3 | 사소한 문제 | 로그 문구 어색함, 아이콘 정렬 문제 |
