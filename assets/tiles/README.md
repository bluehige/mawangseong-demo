# Quarter-View Tile Assets

방 전체 배경 PNG는 이 폴더에 두지 않는다.

- `cave_f/floor`: F급 동굴 바닥 16마스크 타일.
- `cave_f/edge`: 바닥 가장자리와 스커트.
- `cave_f/wall`: 뒤벽/앞벽 세그먼트.
- `cave_f/door`: 문/아치 타일.
- `cave_f/overlay`: 코너, 데칼, 등급 장식 오버레이.
- `lair_e`, `fortress_d`: 이후 등급 테마 확장 자리.

파일명에는 mask, direction, state, frame 정보를 포함한다.
