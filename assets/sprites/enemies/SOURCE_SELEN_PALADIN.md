- Combat role: DAY 15 Chapter 2 boss and Stage 02 examiner
- Required animations: `idle_down`, `move_down`, `attack_down`, `skill_down`, `down`
- Frame contract: 2 idle + 4 move + 4 attack + 4 skill + 2 down = 16 frames. Boss phase logic may request these named clips but must never replace their final frame with an unrelated clip.
- Portrait: `assets/sprites/portraits/onboarding/CHR_SELEN_portrait_checklist.png`
  - Cropped from the built-in image_gen design sheet.

The combat sprite set can be regenerated locally with:

```powershell
python tools\generate_selen_paladin_assets.py
godot --headless --path . --import --quit-after 1
```

Do not use `tools/generate_selen_paladin_assets.py` to overwrite the portrait. The portrait is an imagegen-derived asset and should stay tied to the source sheet above.
