이 파일이 왜 필요한지: 정규버전 BGM/SFX가 늘어날 때 파일 용도를 구분한다.

# Audio Assets

- `bgm`: 챕터, 전투, 이벤트, 엔딩 배경음.
- `sfx`: UI, 전투, 스킬, 방/함정, 보상 효과음.
- `sfx/skills`: 직접 전투 스킬별 고유 발동음. 공통 공격·피격음과 레이어링된다.

v0.5 음악 상태는 관리 화면, 일반 전투, 보스 전투의 세 트랙으로 구분한다. 보스가 전투 중 등장하면 보스 트랙으로 즉시 전환한다.

사용 출처와 생성 방식은 이 폴더 또는 하위 폴더의 `SOURCE.md`에 남긴다.

Lyria 3 후보 생성, 청취 승인, 원본 보존과 런타임 승격 절차는 `docs/audio/LYRIA3_INTERACTIONS_WORKFLOW.md`를 따른다. API 키와 미승인 후보는 저장소에 넣지 않는다.
