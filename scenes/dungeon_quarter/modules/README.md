# Quarter Dungeon Module Scenes

- `rooms`: 입구, 왕좌, 병영, 보물고, 회복실, 건설 슬롯 등 방 모듈.
- `corridors`: 가시 복도, 일반 복도, 함정 복도.
- `junctions`: 중앙 허브, T자/십자 연결 모듈.
- `props`: 모듈 안에 배치되는 장식/차단 오브젝트 후보.

현재 렌더러는 PNG와 데이터 기반으로 동작한다. 씬 모듈은 상호작용, 전경 레이어, 애니메이션 오브젝트가 필요해지는 단계에서 추가한다.
