# Gameplay Systems

정규버전 신규 시스템은 이 폴더 아래 기능별 하위 폴더에 둔다.

- `campaign`: 30일 캠페인, 챕터, 하루 시작/종료.
- `events`: 이벤트 후보 선정, 선택지 결과, 이벤트 로그.
- `economy`: 자원, 보상, 비용, 전리품.
- `progression`: 악명, 마계 명성, 마왕성 등급, 해금.
- `save`: 저장/불러오기, 저장 데이터 버전.
- `tutorial`: 초반 안내와 도움말 조건.
- `content`: 데이터 id 해석, 도감 후보, 콘텐츠 카탈로그.

처음부터 오토로드를 늘리지 않는다. 작은 클래스와 테스트 씬으로 검증한 뒤 `GameRoot` 또는 전용 컨트롤러에 연결한다.
