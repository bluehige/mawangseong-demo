이 파일이 왜 필요한지: 정규버전 콘텐츠 데이터를 코드와 분리해서 확장하기 위한 저장 위치와 규칙을 정한다.

# Regular Version Data

정규버전 신규 콘텐츠 데이터는 이 폴더 아래에 추가한다.

- `campaign`: 30일 진행, 챕터, 하루별 해금, 보스 진입 조건.
- `events`: 메인/랜덤/몬스터/보스/침공 작전 이벤트.
- `monsters`: 일반/희귀 몬스터, 스킬 슬롯, 성장값.
- `enemies`: 침입자, 보스, 웨이브 구성.
- `items`: 전리품, 재료, 설계도, 유물, 소모품, 보물.
- `rooms`: 방, 모듈, 방 효과, 방 해금.
- `traps`: 함정, 발동 조건, 피해/상태 이상.
- `balance`: 난이도, 경제, 전투 보정값.
- `progression`: 악명, 마계 명성, 마왕성 등급, 해금 규칙.
- `localization`: UI/이벤트/도감 텍스트 분리 후보.

현재 데모 호환 데이터는 루트 `data/*.json`에 남겨둔다. 정규 로더가 안정되면 이 폴더로 단계적으로 이관한다.
